# Blogger XML Import Checklist

## Before Import

1. **Backup your blog** — Go to Settings > Other > Back up content > Download
2. **Check XML file size** — Blogger limits imports to ~1MB per file
3. **Verify XML structure** — Ensure `<blog>` root element exists

## Import Steps

1. Go to Blogger Dashboard > Settings > Other
2. Click "Import content" under "Blog tools"
3. Select your XML file
4. Check "Also replace all team member permissions"
5. Click "Import"

## Post-Import Verification

- [ ] All posts imported with correct formatting
- [ ] Images and media display correctly
- [ ] Comments are preserved
- [ ] Labels/tags are applied
- [ ] Page order is correct
- [ ] Custom templates are applied

## Common Issues

| Issue | Solution |
|-------|----------|
| Images not showing | Re-upload via Photos editor |
| Broken formatting | Check HTML in post editor |
| Missing comments | Import via Google Takeout |
| Template not applied | Re-select in Theme settings |

## Tips

- Import during low-traffic hours
- Check mobile preview after import
- Update internal links if URL structure changed
